# ❓ Java Quiz Application

A console-based quiz application built in Java that asks multiple-choice questions and calculates the score.

## 🚀 Features
- Multiple-choice questions
- Score tracking
- Console-based interaction
- Easy to modify questions

## 🛠️ Technologies Used
- Java
- Arrays & Objects
- Scanner class

## ▶️ How to Run
1. Clone the repository
2. Compile:
   javac *.java
3. Run:
   java QuizApp

## 📌 Sample Output
What is the capital of India?  
1. Mumbai  
2. Delhi  
3. Kolkata  
4. Chennai  

Enter your answer: 2
